# Git init - помогает создать пустой репозиторий Git 
 git status -  помогает просмотреть статус нужного репозитория 
git log - помогает просматривать изменения, внесённые в репозиторий
git revert HEAD - помогает откатить последний коммит
git branch new_branch_name - помогает создать новую ветку
git merge existing_branch_name-Объединить две ветки можно параметром merge с указанием имени ветки
git merge origin - Слияние удалённого репозитория с локальным
